# 图书馆管理系统

基于 Vue 3 + Element Plus + Node.js + Express + SQLite 的现代化图书馆管理系统，统一登录入口自动识别角色（学生/管理员），功能完整、开箱即用。

---

## 功能概览

### 学生端功能

| 功能 | 说明 |
|------|------|
| 图书检索 | 支持按书名、作者关键词搜索，按分类（10个分类）筛选 |
| 图书详情 | 查看图书完整信息、封面、库存状态 |
| 图书借阅 | 一键借阅图书，自动更新库存 |
| 图书归还 | 在线归还已借阅图书 |
| AI 智能找书 | 通过自然语言描述找书或获得推荐（如"推荐科幻小说"） |
| 评论评分 | 对图书发表评论和评分，查看他人评价 |
| 个人借阅记录 | 查看自己的借阅历史、当前借阅和归还状态 |

### 管理员端功能

| 功能 | 说明 |
|------|------|
| 数据概览 | 实时统计图书总数、学生人数、借阅记录 |
| 图书管理 | 图书的增删改查，支持上传封面图片 |
| 学生管理 | 查看和管理所有学生账号 |
| 借阅管理 | 查看所有借阅记录，管理借阅状态 |

### 公共功能

| 功能 | 说明 |
|------|------|
| 统一登录 | 学生和管理员共用一个登录入口，自动识别角色跳转 |
| 用户注册 | 学生可自助注册账号 |
| 输入验证 | 学号/电话仅允许数字，密码/邮箱禁止中文 |
| 密码可见 | 支持密码显示/隐藏切换 |
| 密码保护 | 注册密码不直接存储，安全性保障 |

## 技术栈

| 分类 | 技术 |
|------|------|
| 前端框架 | Vue 3 (Composition API + `<script setup>`) |
| UI 组件库 | Element Plus |
| 路由管理 | Vue Router 4 |
| 构建工具 | Vite 5 |
| 后端框架 | Express 4 |
| 数据库 | SQLite 3 (sqlite3) |
| 文件上传 | Multer (封面图片) |
| HTTP 客户端 | Axios |

## 快速开始

### 环境要求

- Node.js >= 18.0.0
- npm >= 9.0.0

### 安装与运行

```bash
# 1. 进入项目目录
cd library-management

# 2. 安装后端依赖并启动
cd backend
npm install
node src/server.js
# 后端运行在 http://localhost:3000

# 3. 新开终端，安装前端依赖并启动
cd ../frontend
npm install
npm run dev
# 前端运行在 http://localhost:5173
```

### 一键启动（推荐）

```bash
# 项目根目录下执行
cd backend && node src/server.js &
cd frontend && npm run dev
```

## API 接口文档

### 认证接口

| 方法 | 路径 | 请求参数 | 描述 |
|------|------|----------|------|
| POST | /api/login | `{ studentId, password }` | 用户登录，自动返回角色类型 |
| POST | /api/register | `{ studentId, name, password, email?, phone? }` | 学生注册 |

### 图书接口

| 方法 | 路径 | 请求参数 | 描述 |
|------|------|----------|------|
| GET | /api/books | `?keyword=&category=` | 获取图书列表（支持搜索和筛选） |
| GET | /api/books/:id | — | 获取单本图书详情 |
| POST | /api/books/:id/comments | `{ content, rating }` | 添加评论和评分 |
| GET | /api/books/:id/comments | — | 获取图书的评论列表 |

### 借阅接口

| 方法 | 路径 | 请求参数 | 描述 |
|------|------|----------|------|
| POST | /api/borrow | `{ bookId, studentId }` | 借阅图书 |
| POST | /api/return | `{ id }` | 归还图书 |
| GET | /api/students/:id/borrow-records | — | 获取学生的借阅记录 |

### 管理员接口

| 方法 | 路径 | 描述 |
|------|------|------|
| GET | /api/admin/stats | 获取统计数据（图书数、学生数、借阅数） |
| GET | /api/admin/books | 获取所有图书 |
| POST | /api/admin/books | 添加图书 |
| PUT | /api/admin/books/:id | 编辑图书 |
| DELETE | /api/admin/books/:id | 删除图书 |
| GET | /api/admin/students | 获取所有学生 |
| POST | /api/admin/students | 添加学生 |
| GET | /api/admin/borrow-records | 获取所有借阅记录 |

### 文件上传接口

| 方法 | 路径 | 请求参数 | 描述 |
|------|------|----------|------|
| POST | /api/upload/cover | `cover` (multipart/form-data) | 上传图书封面，返回图片URL |

### AI 智能找书接口

| 方法 | 路径 | 请求参数 | 描述 |
|------|------|----------|------|
| GET | /api/ai/search | `?query=` | 智能分析搜索意图，返回匹配的图书和推荐建议 |

## 使用说明

### 学生使用流程

1. 打开 http://localhost:5173
2. 输入学号和密码登录（或点击注册标签页创建新账号）
3. 进入学生端后：
   - **图书列表** → 浏览所有图书，按分类筛选或关键词搜索
   - **AI 助手** → 用自然语言描述找书（如"推荐计算机入门"）
   - 点击图书进入详情 → 查看信息、借阅、评论
   - **我的借阅** → 查看借阅记录、归还图书

### 管理员使用流程

1. 使用管理员账号登录（admin / admin123）
2. 进入管理端后：
   - **数据概览** → 查看系统运行数据
   - **图书管理** → 添加/编辑/删除图书，上传封面
   - **学生管理** → 查看学生列表
   - **借阅管理** → 查看所有借阅记录

## AI 智能找书

AI Assistant 功能无需接入外部 API，完全基于内置的智能关键词匹配算法实现：

- **意图分析**：自动分析用户输入的关键词，识别图书分类和难度等级
- **多维评分**：从书名、作者、分类、描述等维度综合匹配
- **匹配原因**：每本推荐图书都附带匹配理由
- **智能建议**：根据搜索内容生成个性化阅读建议

支持的关键词示例：科幻、计算机入门、历史书、小说推荐、哲学经典、心理学书籍等。

## 开发说明

### 数据库

系统使用 SQLite 数据库，首次启动时自动创建表和种子数据。数据库文件位于 `backend/library.db`。如需重置数据，删除该文件后重启服务即可。

### 文件上传

图书封面存储在 `backend/uploads/covers/` 目录，通过 `http://localhost:3000/uploads/covers/` 路径访问。支持 jpg/png/gif/webp 格式，单文件最大 5MB。

### 端口说明

- 前端开发服务器默认使用 5173 端口（被占用时自动递增）
- 后端 API 服务器固定使用 3000 端口
